# Cybersecurity Home Lab

## Project Overview

This home lab was built on macOS using Docker Desktop and OWASP Juice Shop.

The goal of this lab is to practice web application security concepts in a safe and legal environment.

## Technologies Used

- Docker Desktop
- OWASP Juice Shop
- macOS
- Terminal
